# The laplaspack Manifesto

**Memory is context, not results.**

AI can now generate every answer. What it cannot generate is the place your
judgment came from — what you decided, why, and instead of what. Systems that
store what the AI said are building caches. We are building memory.

These are the principles the format stands on.

---

**1. Memory is context, not results.**
A result answers yesterday's question. Context answers tomorrow's. Storing
answers is caching; storing context is remembering. So a pack does not collect
outputs — it keeps the terrain judgments were made on: things, their
properties, and the relations between them.

**2. Structure belongs at write time.**
Intelligence is spent once, when memory is written — so that reading, merging,
and verifying can be deterministic forever after. A memory that must be
re-interpreted by a model on every read is not a record; it is a retelling.

**3. "Why" is a first-class citizen.**
A decision stored without its why cannot be safely changed — it hardens into a
relic. Causal links are how memory knows it has expired: when the reason
falls, the decision asks to be revisited. And the Y is a fork in the road —
record the fork, not only the path taken.

**4. A citation must point at something you can fix.**
If an answer's source cannot be opened and corrected in place, it is not
accountability — it is decoration. Every atom in a pack can be edited, and the
edit is history, not erasure.

**5. Memory is a file you own.**
It can be carried, handed over, inherited, merged — and read with nothing but
a standard library. The escape hatch is not a feature of this format; it is
the reason for it.

**6. Trust is verified, not promised.**
Content is signed. Provenance rides on every atom. Verification needs no
network, no account, and no trust in us. If you must believe the vendor, it is
not your memory.

**7. Machines propose. People decide — once.**
Nothing enters the canon without a human gate, and no identity is ever
guessed. An ambiguity decided once becomes a permanent alias and never asks
again. The gate is not censorship; it is where an output is promoted into
context.

---

We wrote these beliefs down as a file format, because beliefs that live only
in software die with the software. Formats outlive their authors.

`.laplaspack` is our attempt. [Read the spec](./SPEC.md). Break it.
